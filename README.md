# El Paseo Ibérico de Argimiro (Córdoba) 🍷🥩

Proyecto de sitio web multipágina informativo para el restaurante **El Paseo Ibérico de Argimiro** en Córdoba. El objetivo es transmitir la calidad de sus productos ibéricos y facilitar las reservas telefónicas.

## 📝 Registro de decisiones y progreso

### 2026-04-09 | Configuración del Contexto e Identidad Visual
**Acción:** Definición de reglas de código, paleta de colores y tipografía.

**¿Por qué lo hacemos?**
- **Paleta de colores (#5D101E, #F4EBD0, #D4AF37):** Se ha elegido el Burdeos como color principal porque evoca directamente el jamón ibérico y el vino tinto, productos estrella del local. El color crema y el oro viejo aportan calidez y una sensación premium/tradicional.
- **Tipografía (Cinzel / Montserrat):** Usamos **Cinzel** para títulos por su estructura recta y elegante que recuerda a las inscripciones clásicas, reforzando la autoridad y solidez del negocio. **Montserrat** se usa para el cuerpo por su limpieza geométrica y facilidad de lectura.
- **Botones redondeados:** Para suavizar el diseño y hacerlo más moderno y accesible, manteniendo la elegancia con animaciones sutiles.
- **Tecnología Vanilla (HTML/CSS/JS):** Para garantizar un rendimiento óptimo, SEO limpio y control TOTAL sobre el diseño sin dependencias innecesarias de frameworks.
